# Instagram & Messenger Blocker Extension

This is a simple browser extension for Mozilla Firefox that blocks access to Instagram and Messenger. It is built using Python and can be easily modified to block other websites as needed.

## Features
- Blocks Instagram and Messenger on Mozilla Firefox.
- Simple and easy-to-use Python-based extension.
- Customizable to block other websites.
- Lightweight and unobtrusive.

## Installation

1. **Clone this repository** to your local machine:
   ```bash
   git clone https://github.com/yourusername/instagram-messenger-blocker.git

2. **Install** required dependencies:

    Ensure you have Python 3.x installed.
    Install any required packages (if applicable, list them here):

    pip install -r requirements.txt

3. **Load** the extension in Firefox:

    Open Firefox and navigate to about:debugging.
    Click on "This Firefox" and then click on "Load Temporary Add-on".
    Select the manifest.json file from the cloned repository.

4. **Start** using the extension:

    Once loaded, the extension will automatically block access to Instagram and Messenger on your browser.
